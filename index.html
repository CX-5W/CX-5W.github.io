makedown语法

标题

一级标题

#+空格+内容

二级标题

##+空格+内容

接下来以此类推…

列表

无序列表使用星号（*）或者（+）或者（-）作为标记

有序列表使用使用数字加点（.）来表示

  1. 1
  2. 2
  3. 3

分割线

在每一行中用三个的星号、或者减号、或者底线来建立一个分隔符，在这个行里面不能有其他的东西，同时在中间加空格也可以。

123456

13456

321654

链接

写法：This is [an example]+(前面内容的链接)inline link.

This is an example inline link.

加粗

使用星号（*）或者底线（_)作为标记强调字词的符号，被星号或者底线包围的字符会被转成用标签包围，用两个则是强调

这个一个星号的123465

这是两个星号的123456

这是一个底线的_123456_

这是两个底线的__123456__

个人博客搭建教程

准备工作

nodejs

下载链接：https://nodejs.org/zh-cn/download

安装选项说明：

默认勾选了核心组件：Node.js runtime（运行时）、corepack manager（包管理器）、npm

（Node包管理工具）和PATH环境变量配置

核心运行时(node.exe)需要81MB硬盘空间

推荐操作：

新手用户保持默认选项直接点击”Next”即可

高级用户可根据需求：

取消勾选npm（若已单独安装或想使用yarn/pnpm）

调整安装路径（通过”Browse”按钮）

查看磁盘使用情况（”Disk Usage”按钮）

注意事项：

建议保留”Add to PATH”选项以便全局使用Node命令

安装完成后在命令行运行 node -v 和 npm -v 验证安装

自动安装必要工具。注意：此操作会同时安装 Chocolatey。安装完成后，脚本将在新窗口中弹出。
